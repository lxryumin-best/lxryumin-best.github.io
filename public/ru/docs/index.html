Профессиональный опыт#

Независимый ИТ-консультант и архитектор инфраструктуры#

2008 — настоящее время · Ярославль, Россия

Стратегический ИТ-партнёр для 25+ B2B-клиентов из различных отраслей. Ключевые направления работы и достижения:

  • Проектировал и внедрял корпоративные LAN/WAN-инфраструктуры с нуля, включая объект с суммарной длиной кабельной трассы более 4 500 метров UTP
  • Строил отказоустойчивые серверные кластеры на базе Proxmox VE для клиентов с требованием непрерывной работы 24/7
  • Выполнял миграции без простоев — сложные ИТ-переезды и модернизации сетей без единой минуты остановки бизнеса
  • Развёртывал корпоративные решения кибербезопасности (Kaspersky Endpoint Security) в многоплощадочных корпоративных средах
  • Интегрировал VoIP-телефонию, снижая затраты клиентов на связь без привязки к проприетарному оборудованию
  • Обеспечивал SLA-поддержку для 23+ активных корпоративных клиентов (от 3 до 100+ рабочих станций)
  • Проектировал и внедрял решения безопасного удалённого доступа (VPN-туннели, периметр Zero Trust)
